# MultiWipeIt-Nuke-Gizmo

A custom Nuke gizmo that creates automatic multi-stage wipe breakdowns.

## Features

- Unlimited breakdown workflow
- Automatic wipe timing
- One-click setup
- Customizable transition duration
- Easy to integrate into existing scripts

## Installation

Copy MultiWipeIt.gizmo into your Nuke plugins folder.

Then add:
nuke.pluginAddPath("./")

## Usage

1. Connect your renders.
2. Set Start Frame.
3. Choose Transition Length.
4. Play the timeline.
